```java
给定下列程序，那么执行printf("%d\n", foo(20, 13));的输出结果是________。
A 3
B 9
C 27
D 81

正确答案： D

解析：foo(20, 13) = 3 * foo(14, 6) = 3 * 3 * foo(8, 3) = 3 * 3 * 3 * foo(2, 1) = 3 * 3 * 3 * 3 * foo(-4, 0) =3 * 3 * 3 * 3 * 1 = 81
答案：D
```
